# Skill: generate-sentiment-signals

Analyze FX-related news articles to generate bullish/bearish sentiment signals per currency.

## Purpose

Reads daily news JSON files (generated by `aggregate-news`), analyzes sentiment using keyword-based scoring, and outputs directional signals (bullish/bearish/neutral) with confidence scores. These signals form the foundation for the strategy layer to generate pair trade recommendations.

## File Structure

```
/workspace/group/fx-portfolio/data/
  news/{currency}/{date}.json        # Input: aggregated news
  signals/{currency}/{date}.json     # Output: sentiment signals
```

```json
    }
  ]
```


## Sentiment Analysis Methodology

**Current Implementation (Keyword-Based):**

The script counts occurrences of sentiment keywords in the combined news text:

- **Bullish keywords:** gains, rises, climbs, strengthens, hawkish, rate hike, positive, firm, strong
- **Bearish keywords:** falls, declines, tumbles, weakens, dovish, rate cut, negative, soft, weak
- **Neutral keywords:** stable, steady, unchanged, flat, consolidates, mixed, choppy

**Scoring:**
- Bullish score = bullish_count / total_keywords
- Bearish score = bearish_count / total_keywords
- Neutral score = neutral_count / total_keywords

**Direction:** Highest score wins (bullish, bearish, or neutral)

**Confidence:**
- Derived from the winning score
- Boosted by 1.5x (capped at 1.0)
- Higher confidence = clearer sentiment direction

**Strength Levels:**
- Strong: confidence ≥ 0.70
- Moderate: 0.40 ≤ confidence < 0.70
- Weak: confidence < 0.40

## Reasoning Extraction

The script attempts to identify key themes driving sentiment:
- Monetary policy signals (hawkish/dovish)
- Currency strength trends (gains/falls)
- Economic data (positive/weak indicators)

These are included in the `reasoning` field for transparency.

## Key Themes

The generator extracts the top 3 news themes from the most relevant articles based on title patterns:
- Inflation/CPI/PCE data
- GDP/growth data
- Trade policy/tariffs
- Central bank statements
- Currency-specific news

## Improvements

**Current Limitations:**
1. **Keyword-based scoring is simplistic** - doesn't understand context, sarcasm, or nuance
2. **No central bank statement parsing** - misses critical policy signals
3. **No event detection** - doesn't identify major geopolitical events
4. **No source weighting** - treats all sources equally (ForexLive vs FXStreet)
5. **No historical comparison** - doesn't compare today's sentiment vs past trends

**Future Enhancements:**

### 1. LLM-Based Sentiment Analysis (High Priority)
Replace keyword matching with LLM-based analysis:
- Use Claude Haiku for cost-effective sentiment scoring
- Prompt: "Analyze this FX news for {currency}. Is sentiment bullish/bearish/neutral? Rate confidence 0-1. Explain key drivers."
- Benefits: Understands context, detects subtle signals, identifies contradictions
- Cost: ~$0.01 per currency per day (11 currencies × 30 days = $3.30/month)

### 2. Central Bank Statement Parser
- Scrape official statements from Fed, ECB, BoJ, etc.
- Parse for hawkish/dovish language using LLM
- Weight central bank signals higher than general news

### 3. Geopolitical Event Detection
- Identify major events (elections, wars, trade disputes) using LLM
- Flag currencies affected by geopolitical risk
- Adjust confidence based on event magnitude

### 4. Source Weighting
- Assign reliability scores to news sources
- Weight high-quality sources (central banks, Reuters, Bloomberg) higher
- Discount low-quality or contradictory sources

### 5. Historical Sentiment Tracking
- Store 30-day sentiment history per currency
- Compare today's sentiment vs 7-day/30-day average
- Detect sentiment shifts (e.g., "sentiment turning bearish")

### 6. Multi-Timeframe Signals
- Generate intraday signals (1h, 4h) for short-term trading
- Generate weekly/monthly signals for longer-term positioning
- Allow strategy layer to blend timeframes

## Output Validation

**Good Signal Characteristics:**
- ✓ Confidence ≥ 0.50 (moderate or strong)
- ✓ Article count ≥ 5 (sufficient data)
- ✓ Clear reasoning with specific themes
- ✓ Avg relevance ≥ 0.40 (articles are relevant to currency)

**Warning Signs:**
- ⚠️ Confidence < 0.30 (very weak signal, likely noise)
- ⚠️ Article count < 3 (insufficient data)
- ⚠️ Avg relevance < 0.30 (articles not relevant to currency)
- ⚠️ Direction = neutral with low confidence (no clear sentiment)

## Troubleshooting

**All currencies showing bullish:**
- News bias towards "gains" and "rises" in titles
- Increase bearish keyword list or use LLM for balance

**Low confidence scores across the board:**
- Not enough sentiment keywords in news
- Lower minimum threshold or add more keywords
- Consider switching to LLM-based analysis

**Signals don't match market reality:**
- Keyword-based approach misses context
- Implement LLM-based sentiment analysis
- Add central bank statement parsing

**No data for EM currencies (NOK, SEK, CNY, MXN):**
- Add region-specific news sources (see aggregate-news.md)
- Lower relevance threshold for these currencies
- Use web search to find specialist sources
